# 💊 ERP AI Assistant – Medicine Inventory Management

An AI-powered ERP assistant built with Oracle APEX to streamline medicine inventory management within an enterprise environment.

# 📌 Overview
This project was developed during my internship as a practical ERP module for managing pharmaceutical inventory. It allows users to search for medicines by name and instantly retrieve stock levels, inventory details, and expiry information — all through a clean, web-based interface powered by Oracle APEX.

# ✨ Features

🔍 Medicine Search — Look up any medicine by name and instantly view its details
📦 Stock & Inventory Details — Get real-time stock availability and inventory data
📅 Expiry Date Tracking — Monitor expiry dates to support timely stock management
➕ Add New Medicine — Easily register new medicines into the system


# 🛠️ Tech Stack
| Technology | Usage |
|------------|-------|
| Oracle APEX | Frontend & Application Framework |
| Oracle Database | Backend Data Storage |

# 🚀 Getting Started
Prerequisites

- Access to an Oracle APEX workspace
- Oracle Database instance
- Required database privileges to create and manage tables

Setup

- Clone this repository

- bash   git clone https://github.com/your-username/erp-ai-assistant.git

- Import the APEX application export file into your Oracle APEX workspace
Configure the database connection and run the provided SQL scripts to set up the schema
Launch the application from your APEX workspace
